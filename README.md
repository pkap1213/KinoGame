# Kino Game

A Kino-style lottery game built using Java and JavaFX, managed with Maven.

## Description

This project implements a simple Kino game with a JavaFX graphical interface. It is designed to run on modern Java versions and uses Maven for dependency management and execution.

## Requirements

- Java 11 or higher  
- Maven  
- JavaFX (via Maven dependencies or local SDK)

## Setup and Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/pkap1213/KinoGame
   cd https://github.com/pkap1213/KinoGame

2. Running the game:
   ```bash
   mvn compile
   mvn exec:java
